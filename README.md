## Superstore-Analytics-Dashboard
##1. Project Title
**Superstore Analytics Dashboard using Power BI**

## 2. Project Overview
This Power BI project visualizes key sales and profit insights from a Superstore dataset.  
The dashboard enables business users to track performance, analyze product profitability, monitor return rates, and compare year-over-year growth.

## 3. Dataset Description
-Source: Superstore Sales Dataset  
-Time Period: 2020 – 2023
-Key Columns: Order Date, Sales, Profit, Category, Sub-Category, Segment, Region, and Returns
4. Objectives
- Analyze total sales and profit across years  
- Compare performance with the previous year  
- Identify the most profitable products and regions  
- Monitor return percentage and customer segments
5. Key Features
- KPI Cards: Show Sales, Profit, and % Returned Orders with YoY comparison  
- Line Chart: Displays Sales vs. Previous Year trend  
- Bar Chart: Profit by Product sub-category
- Map: Profit by State    
- Donut Chart: Sales by Segment  
- Slicers: Customer Name, Country/Region, Segment, and Date Range
6. Insights
- Sales increased by 47.16% compared to the previous year  
- Profit improved by 48.85%, indicating strong growth  
- Return rate decreased by 2.95%, showing improved customer experience  
- Technology and Copiers are top-performing categories  
- Consumer segment contributes the highest sales (≈50%)
7. Tools & Technologies Used
- Power BI Desktop  
- Power Query for Data Cleaning and Modeling  
- Data Visualization (KPI, Line, Bar, Map, Donut Charts)
